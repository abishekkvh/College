# College

This repository is a curated collection of my college laboratory work, organized semester-wise and written with a focus on clarity, correctness, and hands-on understanding. It is not just a folder of programs; it is a record of concepts being translated into working code.

## Semester 4

### Design & Analysis of Algorithm Laboratory

This section contains C++ implementations of important algorithms studied in the Design and Analysis of Algorithms laboratory. The programs cover searching, sorting, string matching, graph algorithms, greedy techniques, dynamic programming ideas, backtracking, and branch and bound.

Each exercise is kept in its own folder so the repository stays easy to navigate, revise, and extend.

## Programs Included

| Exercise | Programs |
| --- | --- |

| EX2 | Closest Pair, Pattern Matching |
| EX3 | Merge Sort, Quick Sort |
| EX4 | Heap Sort, Insertion Sort |
| EX5 | Horspool String Matching |

The goal of this repository is to keep my academic programming work well-structured, readable, and useful for revision. Every program represents a core algorithmic idea implemented from the ground up, helping connect theory with practical execution.

This repo will continue to grow as more coursework, experiments, and lab programs are added.
